# Guillotine-Plus

A GIMP 3.0 plugin to efficiently slice images into predefined, evenly sized tiles with optional discardable divider lines.

## Target Platform
- **GIMP 3.0+**
- **Python 3**
- **GObject Introspection (GI) API**

## Features
- User-definable tile width and height.
- User-definable divider width (discarded during cut).
- Live preview of cut lines on the canvas.
- Intuitive user interface (auto-generated by GimpProcedureDialog).

## Installation
1. Ensure GIMP 3.0 is installed.
2. Link or copy the `guillotine-plus` folder to your GIMP 3.0 plugins directory:
   - Linux: `~/.config/GIMP/3.0/plug-ins/`
3. Restart GIMP.
4. Find the plugin under `Filters -> Utils -> Guillotine-Plus`.

## Development
This project uses Git for version control and a standard Python structure.
The core logic resides in `lib/` and the main plugin entry point is `guillotine-plus/guillotine-plus.py`.
